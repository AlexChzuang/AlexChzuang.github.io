GitHub 的小小客製

2021-05-27
#客製化
GitHub

GitHub 是個十分好用的版控管理工具，自從開始學寫程式以後，變得跟它密不可分，像極了愛情 💌

從沒想過原來個人首頁也可以客製化，直到發現班上某位同學的 GitHub 長得不太一樣，決定來小研究一下～

今天要分享的，是在 GitHub 的個人首頁放上自訂的招呼語 👋

第一步：新增一個新的 Repo

第二步：用自己的帳號命名，就會跳出一個神秘的視窗 ✨
告訴你成功找到了一個神奇的功能（還在旁邊歡呼）

第三步：別忘了新增一個 README 檔案
（這邊忘記其實也沒關係，事後也可以再加）

大功告成！這樣就可以直接修改這份 README 檔，放上想讓大家知道的資訊囉！
