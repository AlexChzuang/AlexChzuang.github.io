『三個月後，我是軟體工程師。』（好像什麼ＯＸ廣吿斯露梗）

今天又是HTML/CSS的課程，相較之下理解能力會高一些，也比較有成就感一點，畢竟這可以當下就看到敲出指令後的成果，當然希望未來進職場後，也可以獲得滿滿的成就感～

中午也和未曾聊過的同學聊天，每個人都有屬於自己一段特別的經歷，所以才會說：「一樣米養百樣人」吧。（可以這樣用？）

「誠實面對自己的學習狀況。」是老師一直強調的點；當然這是十分重要的。『寧願在課堂上問蠢問題被洗臉，也不要之後工作找得灰頭土臉。』解決問題才是關鍵！

下面來分享一下今天的小小小成果：

第一張圖：建立四個欄位，上方擺圖，下方擺標題與簡短介紹。

display: flex / flex-wrap / text-align / div class wrap /div class col / img src /


左邊這張圖：是上面那張的前兩個練習，建立一個div box，並設定 width, font-size, border solid, margin & padding.

以上是簡短的練習分享，真的是蠻有趣的。原本在內文的地方都是用lorem 新增亂數的英文；今天學到中文的亂數：ctlorem。直翻應該是代表chinese tradition吧。所以你可以看到右邊滿滿的字，毫無意義哈哈哈。

希望我可以繼續持之以恆，不過我覺得第二天目前還蠻不錯的，也可以順便學習的成長紀錄？
